# MASTER TZ SUMMARY

AOPMem v0.1 is a macOS ARM Rust CLI product developed in its own repo.

It installs globally to `~/.aopmem` and initializes workspaces in `~/.aopmem/workspaces/<workspace-key>`.

AOPMem stores agent operational memory in SQLite with FTS5/BM25 and typed graph links.

It does not use Mem0, Hindsight, semantic search, vector search, Qdrant, custom MCP server, CI, markdown memory export/import, migration, current state, task history, QA domain pack, or PR/handoff pack.

Development must follow DERC and stage prompts.
